Flask Tutorial

There are many languages and web frameworks to choose them, so why should you pick flask? Flask is a simple, yet extensible, web framework written in python. It is easy to get a project started, but there are many features you can add as you can go. I used flask to build this website, and it was a great framework to start with.

This guide goes over installing Python, Flask, Gunicorn, and utilizes Github and Heroku for deployment.

1. Install Python

If you are running Mac or Linux/BSD you should have python installed already. These installation instructions are currently for Mac users, but the process is similar for those on Linux/BSD. Python 2 has been the standard, but Python 3 is coming around. Python 3 has some changes that are not backwards compatible. Flask is currently only support on Python 2, plus many packages are still dependent on Python 2, so you should stick with that when using Flask. Using virtualenv is also highly suggested, as this sandboxes each environment separately allowing you to install packages on a per project basis allowing for portability. I do not currently go through that here, but it is relatively easy to setup. The following code should work on most systems.

Install command line tools:

xcode-select --install

If you have XThis does install a version of python which you can use, but if you want to use a newer version, I suggest using homebrew.

Install Homebrew:

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Run this command to verify homebrew is installed properly:

brew doctor

If any problems occur, follow the instructions and you should be on your way.

And finally, install Python:

brew install python

Newer versions of Python come with pip, the python package manager, installed by default.

2. Install Flask

Install flask & virtualenv & gunicorn with pip:

pip install flask virtualenv gunicorn

This will install Flask and its dependencies, along with virtualenv and Gunicorn, a web server.

3. Building your app

Now this is where the real fun begins!

For starters, we need to create a python file to write our app. Create a file called 'app.py' with your favorite text editor and insert the following code:


    from flask import Flask, render_template

    app = Flask(__name__)

    @app.route('/')
    def home():
      return render_template('home.html')

    if __name__ == '__main__':"
      app.run()

Let's go through what this code does. First, you import flask, then you import render_template which will render our html files. We then initiliaze a new flask app called 'app'. Next, we create a root route and tell it to render our 'home.html' file. We will create that next.
